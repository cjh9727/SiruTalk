# Android messenger "Siru-Talk"
---
시루톡
---


시루톡은 사용자가 보낸 메시지의 잘못된 맞춤법을 자동으로 인식 및 수정하여 사용자의 올바른 맞춤법 사용을 돕고, 부가적으로 사용자간의 원활한 의사소통을 이끌어내는 메신저이다.
